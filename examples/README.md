# Examples

End-to-end recipes showing how Helix is meant to be used. (Illustrative during pre-alpha —
they describe the target experience; runnable versions land as phases ship.)

| Recipe | Shows |
|---|---|
| [`quickstart.py`](quickstart.py) | Remember + recall locally via the Python SDK ($0, offline) |
| `connect-claude-code.md` *(soon)* | One-command MCP wiring so Claude Code recalls your memory |
| `transfer-between-laptops.md` *(soon)* | `export` a `.dna`, move it, `import` on a new machine |
| `team-merge.md` *(soon)* | Two teammates `merge` strands without leaking secrets |

See [`docs/MCP_INTEGRATION.md`](../docs/MCP_INTEGRATION.md) for the agent-facing surface and
[`ROADMAP.md`](../ROADMAP.md) for what's live.
